---
title: 'Mint NFT'
api: 'POST /api/nft/mint'
description: 'Mint a permanent NFT on Arweave blockchain'
---

## Request

<ParamField body="assetId" type="string" required>
  The ID of the uploaded asset to mint as NFT
</ParamField>

<ParamField body="blockchain" type="string" required>
  Target blockchain (currently only "arweave" is supported)
</ParamField>

<ParamField body="walletAddress" type="string" required>
  The wallet address that will own the NFT
</ParamField>

<ParamField body="metadata" type="object" required>
  NFT metadata object
  
  <Expandable title="metadata properties">
    <ParamField body="name" type="string" required>
      Name of the NFT
    </ParamField>
    
    <ParamField body="description" type="string">
      Description of the NFT
    </ParamField>
    
    <ParamField body="attributes" type="array">
      Array of trait objects
      
      ```json
      [
        {
          "trait_type": "Creator",
          "value": "Artist Name"
        }
      ]
      ```
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="royaltyPercentage" type="number">
  Royalty percentage for secondary sales (default: 10)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the mint was successful
</ResponseField>

<ResponseField name="nft" type="object">
  The minted NFT data
  
  <Expandable title="nft properties">
    <ResponseField name="id" type="string">
      Internal NFT ID
    </ResponseField>
    
    <ResponseField name="arweaveId" type="string">
      Arweave transaction ID (manifest ID)
    </ResponseField>
    
    <ResponseField name="arweaveUrl" type="string">
      Permanent URL to the NFT on Arweave
    </ResponseField>
    
    <ResponseField name="metadataUrl" type="string">
      URL to the NFT metadata JSON
    </ResponseField>
    
    <ResponseField name="assetUrl" type="string">
      URL to the actual asset file
    </ResponseField>
    
    <ResponseField name="cost" type="number">
      Total cost in AR tokens
    </ResponseField>
    
    <ResponseField name="blockchain" type="string">
      Blockchain used (arweave)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

## Errors

<ResponseField name="400" type="error">
  **Bad Request** - Missing required fields or invalid data
</ResponseField>

<ResponseField name="404" type="error">
  **Not Found** - Asset does not exist
</ResponseField>

<ResponseField name="500" type="error">
  **Server Error** - Arweave wallet not configured or minting failed
</ResponseField>

## Notes

<Info>
  **Permanent Storage**: Once minted, your NFT is stored permanently on Arweave for 200+ years. This action cannot be reversed.
</Info>

<Warning>
  **Cost**: You will be charged based on the file size. Use the `/api/nft/estimate` endpoint to get a cost estimate before minting.
</Warning>

<Tip>
  **Verification**: After minting, verify your NFT on [Arweave Explorer](https://viewblock.io/arweave) using the `arweaveId`.
</Tip>
